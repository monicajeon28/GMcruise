**구조**:
```

  /partner/[managerId]/mall
    ├─ 랜딩페이지
    ├─ 상품 목록
    ├─ 고객 관리
    └─ 판매 관리

판매원 개인몰:
  /partner/[agentId]/mall
    ├─ 랜딩페이지
    ├─ 상품 목록
    ├─ 고객 관리
    └─ 판매 관리
```

**특징**:
- ✅ 완전 독립적
- ✅ 개인 브랜딩
- ✅ 각자 관리
- ✅ 최적화 없음 (현재 구조 유지)

**완료된 기능**:
- ✅ WebP 자동 변환
- ✅ 이미지 갤러리
- ✅ 소스코드 복사
- ✅ 폴더별 정리

- ⏸️ 판매원/점장 공유 링크

```

  ├─ 네이버 아카데미 스타일
  ├─ 썸네일 기반 선택
  ├─ 카테고리별 분류
```

**구조**:
```
/partner/[agentId]/dashboard
  ├─ 나만의 SNS 마케팅 링크

```
