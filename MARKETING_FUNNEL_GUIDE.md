# 마케팅 퍼널 연결 가이드

> 크루즈 마케팅 광고 랜딩페이지를 자유롭게 만들고 퍼널로 연결하는 방법 🎯

---

## 🎯 목적

**리드젠(Leadgeny) 스타일의 마케팅 퍼널 시스템**

- 유입랜딩 → 설문 → 땡큐 페이지처럼 순서대로 연결
- 고객들이 못보는 비공개 마케팅 페이지 생성
- 퍼널을 자유롭게 만들어서 고객 이동시키기

---

## 📋 사용 예시

### 예시 1: 유입랜딩 → 설문 → 땡큐 퍼널

```
1. 유입랜딩 페이지 (funnelOrder: 1)
   - URL: https://leadgeny.kr/i/yjo
   - 공개: ✅ (고객이 볼 수 있음)
   - 목적: 고객 유입

2. 설문 페이지 (funnelOrder: 2)
   - URL: https://leadgeny.kr/i/yrZ
   - 공개: ❌ (비공개 - 마케팅용)
   - 목적: 고객 정보 수집

3. 땡큐 페이지 (funnelOrder: 3)
   - URL: https://leadgeny.kr/i/yjq
   - 공개: ✅ (고객이 볼 수 있음)
   - 목적: 감사 메시지
```

---

## 🚀 사용 방법

### 1. 관리계정 만들기

1. `/admin/marketing/accounts` 페이지로 이동
2. "계정 생성" 버튼 클릭
3. 계정 이름과 코드 입력
4. 저장

### 2. 퍼널 만들기

1. `/admin/marketing/funnels` 페이지로 이동
2. "퍼널 생성" 버튼 클릭
3. 퍼널 이름 입력 (예: "크루즈 여행 퍼널")
4. 관리계정 선택
5. 저장

### 3. 랜딩 페이지 만들기

1. `/admin/landing-pages/new` 페이지로 이동
2. 페이지 정보 입력:
   - 제목: "유입랜딩 페이지"
   - HTML 내용: 마케팅 콘텐츠 작성
   - **마케팅 계정**: 위에서 만든 계정 선택
   - **퍼널**: 위에서 만든 퍼널 선택
   - **퍼널 순서**: 1 (첫 번째 페이지)
   - **공개 여부**: ✅ 공개 (고객이 볼 수 있음)

3. 저장

### 4. 다음 페이지 연결하기

1. 같은 퍼널에 두 번째 페이지 생성:
   - 제목: "설문 페이지"
   - **퍼널**: 같은 퍼널 선택
   - **퍼널 순서**: 2 (두 번째 페이지)
   - **공개 여부**: ❌ 비공개 (고객이 못봄 - 마케팅용)

2. 세 번째 페이지 생성:
   - 제목: "땡큐 페이지"
   - **퍼널**: 같은 퍼널 선택
   - **퍼널 순서**: 3 (세 번째 페이지)
   - **공개 여부**: ✅ 공개

---

## 🔗 퍼널 연결 확인

1. `/admin/marketing/pages` 페이지로 이동
2. 퍼널별로 그룹화되어 표시됨
3. 각 퍼널의 페이지들이 순서대로 표시됨 (1 → 2 → 3)

---

## 🔒 비공개 페이지 설정

**고객들이 못보는 마케팅 랜딩페이지**

- `isPublic: false`로 설정하면 비공개 페이지가 됩니다
- 비공개 페이지는 일반 고객이 접근할 수 없습니다
- 마케팅 퍼널 내에서만 사용됩니다

**사용 예시:**
- 설문 페이지 (고객 정보 수집용)
- 중간 단계 페이지 (퍼널 내부 전용)
- A/B 테스트 페이지

---

## 📊 퍼널 관리

### 퍼널별 페이지 확인
- `/admin/marketing/pages`에서 퍼널별로 그룹화되어 표시
- 각 퍼널의 페이지들이 순서대로 표시됨

### 퍼널 수정
- `/admin/marketing/funnels`에서 퍼널 정보 수정
- 퍼널 이름, 상태 변경 가능

### 페이지 순서 변경
- 각 페이지의 `funnelOrder` 값을 변경하여 순서 조정
- 예: 1 → 2 → 3 순서로 고객이 이동

## 💡 실제 사용 시나리오

### 시나리오 1: 크루즈 여행 상품 홍보

```
1. 유입랜딩 (공개)
   - 광고 클릭 → 이 페이지로 이동
   - 크루즈 여행 소개
   - "상세 정보 보기" 버튼

2. 상세 정보 (비공개)
   - 고객 정보 수집
   - 설문 조사
   - "신청하기" 버튼

3. 신청 완료 (공개)
   - 감사 메시지
   - 다음 단계 안내
```

### 시나리오 2: 이벤트 참여 유도

```
1. 이벤트 소개 (공개)
   - 이벤트 내용 소개
   - "참여하기" 버튼

2. 참여 정보 입력 (비공개)
   - 개인정보 수집
   - 참여 조건 확인

3. 참여 완료 (공개)
   - 참여 확인 메시지
   - 당첨 안내
```
