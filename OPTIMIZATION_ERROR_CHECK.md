## 🔍 발견된 문제 및 수정

### 1. recharts 동적 임포트 문제 ❌ → ✅ 수정 완료

**문제**:
- recharts 컴포넌트를 개별적으로 동적 임포트 시도
- JSX 컴포넌트는 개별 함수로 동적 임포트 불가

**수정**:
- 차트 컴포넌트를 별도 파일로 분리 (`components/admin/DashboardCharts.tsx`)
- 전체 차트 컴포넌트를 동적 임포트로 변경
- 로딩 상태 추가

**Before** (잘못된 방식):
```typescript
const LineChart = dynamic(() => import('recharts').then(mod => mod.LineChart), { ssr: false });
```

**After** (올바른 방식):
```typescript
const DashboardCharts = dynamic(
  () => import('@/components/admin/DashboardCharts'),
  { ssr: false, loading: () => <div>로딩 중...</div> }
);
```
