# 판매원/대리점장 여권 요청 링크 사용 가이드

**작성일**: 2025년 1월 28일  
**대상**: 판매원, 대리점장

---

## 📍 접근 방법

### 1. 로그인
- 판매원/대리점장 로그인 페이지에서 로그인
- 로그인 후 자동으로 본인의 대시보드로 이동

### 2. 여권 요청 관리 페이지 접근
**URL**: `/partner/[본인의mallUserId]/passport-requests`

**접근 경로**:
1. 대시보드 메인 화면에서
2. 좌측 메뉴 또는 상단 메뉴에서 **"여권 요청 관리"** 클릭
3. 또는 직접 URL 입력: `https://www.cruisedot.co.kr/partner/[본인의mallUserId]/passport-requests`

---

## 🎯 기능 개요

여권 요청 관리 페이지에서는 고객에게 여권 제출 링크를 생성하고 전달할 수 있습니다.

### 제공되는 기능
1. **여권 요청 링크** (빠른 링크 생성) - 새로 추가됨 ✅
2. **링크 생성** (수동 입력 폼) - 기존 기능 유지 ✅

---

## 📋 사용 방법

### 방법 1: 여권 요청 링크 (빠른 링크 생성) ⚡

**언제 사용하나요?**
- 빠르게 링크를 생성하고 싶을 때
- 기본 메시지로 충분할 때
- 템플릿 수정이 필요 없을 때

**사용 단계**:

1. **여권 요청 관리 페이지 접근**
   - `/partner/[본인의mallUserId]/passport-requests`

2. **고객 목록 확인**
   - 테이블에서 여권 요청이 필요한 고객 찾기
   - 상태가 "대기중"인 고객만 링크 생성 가능

3. **"여권 요청 링크" 버튼 클릭**
   - 고객 행의 작업 열에서 초록색 **"여권 요청 링크"** 버튼 클릭
   - 버튼 위치: 테이블의 가장 오른쪽 "작업" 열

4. **모달에서 링크 확인**
   - 모달이 열리면 자동으로 생성된 링크와 메시지가 표시됩니다
   - 기본 메시지 형식:
     ```
     안녕하세요 [고객명]님,

     여권 정보 제출을 위해 아래 링크를 클릭해주세요.

     [여권 제출 링크]

     감사합니다.
     ```

5. **메시지 수정 (선택사항)**
   - 텍스트 영역에서 메시지 내용을 수정할 수 있습니다
   - 링크는 자동으로 포함되어 있습니다

6. **복사 및 전송**
   - **"복사"** 버튼: 링크만 복사
   - **"메시지 복사"** 버튼: 링크가 포함된 전체 메시지 복사
   - 복사한 내용을 카카오톡, 문자, 이메일 등으로 고객에게 전송

7. **완료**
   - 모달 닫기
   - 고객 목록에서 해당 고객의 상태가 "확인 중"으로 변경됨

---

### 방법 2: 링크 생성 (수동 입력 폼) 📝

**언제 사용하나요?**
- 템플릿을 선택하고 싶을 때
- 메시지를 직접 작성하고 싶을 때
- 링크 만료 시간을 설정하고 싶을 때 (최대 14일)

**사용 단계**:

1. **여권 요청 관리 페이지 접근**
   - `/partner/[본인의mallUserId]/passport-requests`

2. **고객 목록에서 "링크 생성" 버튼 클릭**
   - 파란색 **"링크 생성"** 버튼 클릭

3. **템플릿 선택**
   - 드롭다운에서 사용할 템플릿 선택
   - 기본 템플릿이 자동으로 선택됩니다

4. **메시지 수정**
   - 템플릿 내용이 자동으로 입력됩니다
   - 필요에 따라 메시지 내용 수정
   - 사용 가능한 변수:
     - `{고객명}`: 고객 이름
     - `{링크}`: 여권 제출 링크 (자동 치환)
     - `{상품명}`: 상품명
     - `{출발일}`: 출발일

5. **링크 만료 시간 설정 (선택사항)**
   - 기본값: 72시간 (3일)
   - 최대: 336시간 (14일)
   - 최소: 1시간

6. **"링크 생성하기" 버튼 클릭**
   - 링크와 메시지가 생성됩니다

7. **결과 확인 및 복사**
   - 생성된 메시지와 링크가 표시됩니다
   - **"메시지 복사"** 또는 **"링크 복사"** 버튼으로 복사
   - 고객에게 전송

## 💡 유용한 팁

### 1. 검색 기능 활용
- 고객명 또는 전화번호로 빠르게 고객 찾기
- 검색창에 입력하면 실시간으로 필터링됩니다

### 2. 상태 확인
- **대기중**: 아직 여권 요청을 보내지 않은 고객
- **확인 중**: 여권 요청을 보냈지만 아직 제출하지 않은 고객
- **완료**: 여권 제출을 완료한 고객

### 3. 재요청 가능
- "여권 요청 링크"는 이미 요청한 고객에게도 다시 생성 가능
- 링크가 만료되었거나 고객이 링크를 잃어버린 경우 재요청

### 4. 메시지 커스터마이징
- "여권 요청 링크"로 생성한 후에도 모달에서 메시지 수정 가능
- 고객별로 맞춤 메시지 작성 가능

## ❓ 자주 묻는 질문 (FAQ)

### Q1. "여권 요청 링크"와 "링크 생성"의 차이는?
**A**: 
- **여권 요청 링크**: 빠르게 기본 메시지와 함께 링크 생성 (1클릭)
- **링크 생성**: 템플릿 선택, 메시지 수정, 만료 시간 설정 등 상세 설정 가능

### Q2. 링크는 얼마나 유효한가요?
**A**: 
- **여권 요청 링크**: 기본 72시간 (3일)
- **링크 생성**: 1시간~336시간 (14일) 중 선택 가능

### Q3. 이미 요청한 고객에게도 다시 보낼 수 있나요?
**A**: 네, 가능합니다. "여권 요청 링크" 버튼은 항상 표시되며, 재요청이 가능합니다.

### Q4. 생성한 링크는 어디에 저장되나요?
**A**: 
- 링크는 시스템에 자동으로 저장됩니다
- 고객이 링크를 통해 여권을 제출하면 상태가 "완료"로 변경됩니다
- 링크는 복사해서 카카오톡, 문자 등으로 직접 전송해야 합니다

### Q5. 템플릿은 어디서 관리하나요?
**A**: 템플릿은 관리자 패널에서 관리되며, 판매원/대리점장은 선택만 할 수 있습니다.
