### 1. 교육 페이지 (네이버 아카데미 스타일)

```
/education 또는 /admin/education
  ├─ 헤더
  │   ├─ 검색
  │   └─ 카테고리 필터
  ├─ 메인
  │   ├─ [마케팅 교육]
  │   │   ├─ 썸네일 1: "SNS 마케팅 기초"
  │   │   ├─ 썸네일 2: "고객 확보 전략"
  │   │   └─ 썸네일 3: "성공 사례 분석"
  │   │
  │   ├─ [세일즈 교육]
  │   │   ├─ 썸네일 1: "크루즈 상품 이해"
  │   │   ├─ 썸네일 2: "고객 응대 매뉴얼"
  │   │   └─ 썸네일 3: "판매 기법"
  │   │
  │   └─ [인솔 교육] (크루즈스탭용)
  │       ├─ 썸네일 1: "크루즈 기본 정보"
  │       ├─ 썸네일 2: "인솔 가이드"
  │       └─ 썸네일 3: "응급 상황 대응"
  │
  └─ 교육 상세 페이지
      ├─ 비디오 플레이어
      ├─ 강의 자료 다운로드
      ├─ 수강 완료 체크
      └─ 다음 강의 추천
```

**특징**:
- ✅ 썸네일 기반 선택
- ✅ 카테고리별 분류
- ✅ 역할별 맞춤 교육
- ✅ 수강 진행률 추적

### 2. 개인몰 최적화 구조

#### 현재 구조 (최적화 전)
```
판매원 개인몰:
  ├─ 랜딩페이지 (복잡한 디자인)
  ├─ 상품 목록
  ├─ 고객 관리
  └─ 판매 관리
```

#### 최적화 후 구조
```
판매원 개인몰:
  ├─ 간단한 랜딩페이지 (최소화) ✅
  │   ├─ 기본 정보만
  │   ├─ SNS 링크
  │   └─ 연락처
  │
  ├─ 개인 대시보드 (필수) ✅
  │   ├─ 나만의 고객 관리
  │   ├─ 나만의 판매 관리
  │   ├─ SNS 마케팅 링크
  │   └─ 교육 페이지 링크
  │
  └─ 개인 SNS 링크 (필수) ✅
      ├─ 랜딩페이지 링크
      ├─ 상품 공유 링크
      └─ 추적 코드
```

**최적화 포인트**:
- ✅ 랜딩페이지: 최소한의 정보만 (이름, 연락처, SNS)
- ✅ 대시보드: 모든 기능은 대시보드에서
- ✅ SNS 링크: 마케팅용 링크는 유지
