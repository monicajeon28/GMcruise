# 관리계정 관리 기능 설명

> 초등학생도 이해할 수 있게 쉽게 설명해드려요! 🎓

---

## 🤔 관리계정이 뭐예요?

**관리계정**은 여러 개의 마케팅 계정을 만들 수 있는 기능이에요!

### 쉽게 말하면:
- 여러 회사를 관리할 때 각 회사마다 계정을 만들어요
- 예: "A회사 계정", "B회사 계정", "C회사 계정"
- 각 계정마다 고객, 페이지, 퍼널을 따로 관리할 수 있어요

### 왜 필요한가요?
- **회사가 여러 개일 때**: 각 회사마다 고객을 따로 관리하고 싶어요
- **프로젝트가 여러 개일 때**: 각 프로젝트마다 마케팅을 따로 하고 싶어요
- **팀이 여러 개일 때**: 각 팀마다 독립적으로 마케팅을 하고 싶어요

---

## 📊 프리미엄 플랜 사양

- **최대 계정 수**: 10개
- **각 계정당 최대 고객 수**: 9,999명
- **각 계정당 최대 페이지 수**: 200개
- **각 계정당 최대 퍼널 수**: 40개

---

## 🎯 어떻게 사용하나요?

### 1. 계정 만들기
1. "계정 생성" 버튼을 눌러요
2. 계정 이름을 입력해요 (예: "A회사 계정")
3. 계정 코드를 입력해요 (예: "company-a")
4. 소유자를 선택해요
5. 저장하면 끝!

### 2. 계정 관리하기
- **수정**: 각 행의 수정 버튼을 눌러요
- **삭제**: 각 행의 삭제 버튼을 눌러요
- **상태 변경**: 활성/일시정지/삭제 상태로 변경할 수 있어요

### 3. 사용량 확인하기
- 각 계정마다 현재 사용량을 볼 수 있어요
- 예: "고객: 1,234 / 9,999명"
- 예: "페이지: 50 / 200개"
- 예: "퍼널: 10 / 40개"

---

## 💡 실제 사용 예시

### 예시 1: 여러 회사 관리
```
회사 A 계정
  - 고객: 5,000명
  - 페이지: 100개
  - 퍼널: 20개

회사 B 계정
  - 고객: 3,000명
  - 페이지: 50개
  - 퍼널: 15개
```

### 예시 2: 여러 프로젝트 관리
```
프로젝트 A 계정 (크루즈 여행)
  - 고객: 2,000명
  - 페이지: 30개
  - 퍼널: 10개

프로젝트 B 계정 (호텔 예약)
  - 고객: 1,500명
  - 페이지: 25개
  - 퍼널: 8개
```

---

## ⚠️ 주의사항

1. **최대 계정 수**: 프리미엄 플랜은 최대 10개까지만 만들 수 있어요
2. **계정 삭제**: 계정을 삭제하면 그 계정의 모든 데이터(고객, 페이지, 퍼널)도 함께 삭제돼요
3. **계정 코드**: 계정 코드는 중복될 수 없어요 (고유해야 해요)

---

## 🔗 다른 기능과의 연결

### 고객 관리
- 각 계정마다 고객을 따로 관리해요
- 고객 추가할 때 어떤 계정에 추가할지 선택해요

### 퍼널 관리
- 각 계정마다 퍼널을 따로 만들어요
- 퍼널 만들 때 어떤 계정에 만들지 선택해요

### 랜딩 페이지
- 각 계정마다 랜딩 페이지를 따로 만들어요
- 페이지 만들 때 어떤 계정에 만들지 선택해요
