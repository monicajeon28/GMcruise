# 세일즈 채팅봇 업그레이드 계획

## 목표
1. 각 상품별로 독립적인 세일즈봇 플로우 생성
2. 대화형 AI로 플로우 생성 (채팅 인터페이스)
3. 고객에게 링크로 전달 가능한 공개 세일즈봇
4. 효율적인 플로우 관리 (복사, 템플릿)

## 구현 계획

### Phase 1: 스키마 업데이트 (최우선)
- ChatBotFlow에 필드 추가:
  - `productCode String?` - 특정 상품에 연결된 플로우
  - `shareToken String? @unique` - 공개 링크용 토큰
  - `isPublic Boolean @default(false)` - 공개 여부
  - `createdBy Int?` - 생성자 ID

### Phase 2: 상품별 플로우 생성 기능
- 관리자 패널에서 상품 선택 후 플로우 생성
- 기존 플로우를 템플릿으로 복사 가능
- 상품 정보 자동 반영

### Phase 3: 대화형 AI 플로우 빌더
- 채팅 인터페이스로 플로우 생성
- "이런 플로우 만들어줘" → AI가 질문 생성
- 실시간 수정 및 미리보기

### Phase 4: 공개 링크 생성
- shareToken 기반 공개 링크 생성
- `/chat-bot/share/{token}?productCode=XXX` 형식
- 로그인 없이 접근 가능

### Phase 5: 플로우 복사/템플릿
- 기존 플로우를 복사하여 새 상품에 적용
- 템플릿 저장 및 재사용

## 효율성 고려사항
- 상품별 플로우는 선택적 (없으면 기본 플로우 사용)
- 공개 링크는 shareToken으로 관리 (보안)
- AI 빌더는 점진적 개선 (기본 기능부터)
