# 랜딩페이지 관리 기능 완성도 가이드

## ✅ 구현 완료된 기능

### 1. 랜딩페이지 자유롭게 만들기
- ✅ 랜딩페이지 생성
- ✅ HTML 콘텐츠 편집
- ✅ 그룹별 관리
- ✅ 카테고리 분류
- ✅ 관리자 공유 템플릿을 내 계정으로 복사·수정

### 2. 신규 링크 만들기 (바로가기 URL)
- ✅ 짧은 URL 생성 (`/i/코드`)
- ✅ 리드젠 스타일: `https://leadgeny.kr/i/yjo` 형식
- ✅ 신규생성 버튼으로 재생성 가능
- ✅ 카카오톡 캐시 문제 해결을 위한 재생성 기능

### 3. 링크 복사하기
- ✅ 일반 URL 복사
- ✅ 바로가기 URL 복사
- ✅ 클립보드 자동 복사

### 4. 랜딩페이지 수정하기
- ✅ 제목 수정
- ✅ HTML 콘텐츠 수정
- ✅ 그룹 변경
- ✅ 카테고리 변경

### 5. 랜딩페이지 삭제하기
- ✅ 삭제 확인 모달
- ✅ 안전한 삭제 기능

### 6. 관리자 공유 랜딩페이지 커스터마이징
- ✅ 관리자 패널에서 전달된 랜딩페이지 목록 표시
- ✅ “내 페이지로 복사” 버튼으로 즉시 복제
- ✅ 복사 직후 편집 화면으로 이동해 이름/연락처 등 정보수집 필드와 고객 그룹을 직접 설정
- ✅ 복사한 랜딩페이지는 내 쿼터(최대 15개)에 포함되며 자유롭게 SMS/자동화 흐름에 연결 가능

### 6. 그룹별 관리
- ✅ 그룹별로 랜딩페이지 표시
- ✅ 그룹 펼치기/접기
- ✅ 그룹 내 페이지 개수 표시

### 7. 조회수 확인
- ✅ 각 페이지별 조회수 표시
- ✅ 데이터 확인 버튼

### 8. 접속현황
- ✅ 접속현황 페이지로 이동

## 🎯 사용 방법

## 🚫 제한 사항 (2025-11 업데이트)
- 대리점장 대시보드의 새 랜딩페이지 제작 도구에서는 `상품 구매 기능 사용` 토글과 결제 설정 UI가 제거되었습니다.
- 페이앱·웰컴페이먼츠 등 결제 연동이 필요한 경우 별도의 결제 전용 랜딩페이지나 관리자 채널을 이용해야 합니다.
- 기존에 생성된 랜딩페이지에서 수집한 결제 정보는 영향을 받지 않지만, 새 랜딩페이지에서는 결제 구성을 추가할 수 없습니다.

### 1. 랜딩페이지 만들기

1. `/admin/landing-pages` 페이지로 이동
2. "새 랜딩페이지" 버튼 클릭
3. 페이지 정보 입력:
   - 제목
   - 카테고리
   - 페이지 그룹 (예: "16 B2B크루즈")
   - HTML 콘텐츠
4. 저장

### 2. 바로가기 URL 만들기

1. 랜딩페이지 목록에서 "공유하기" → "URL 복사" 버튼 클릭
2. 바로가기 URL 모달이 열림
3. "신규생성" 버튼 클릭하여 새 URL 생성
4. "복사" 버튼 클릭하여 URL 복사

**생성되는 URL 형식:**
```
https://yourdomain.com/i/abc123
```

### 3. 링크 복사하기

**일반 URL 복사:**
- "링크복사" → "URL 복사" 버튼 클릭
- `/landing/페이지-slug` 형식

**바로가기 URL 복사:**
- "공유하기" → "URL 복사" 버튼 클릭
- 모달에서 "복사" 버튼 클릭
- `/i/코드` 형식

### 4. 랜딩페이지 수정하기

1. 랜딩페이지 목록에서 "컨텐츠관리" → 수정 아이콘 클릭
2. 페이지 정보 수정
3. 저장

### 6. 관리자 공유 랜딩페이지 활용하기

1. `/partner/[partnerId]/landing-pages` 페이지 하단의 **관리자 보너스 랜딩페이지** 섹션에서 원하는 템플릿을 찾습니다.
2. 해당 행에서 **내 페이지로 복사** 버튼을 클릭합니다.
3. 복사에 성공하면 자동으로 편집 화면으로 이동합니다.
4. 이름, 연락처 등 정보수집 필드를 활성화하거나 필수로 지정하고, 고객 그룹을 선택하여 수집된 리드를 자동으로 이동시킵니다.
5. 저장 후에는 일반 랜딩페이지와 동일하게 문자 발송, 그룹 자동 이동, 퍼널 연동 등을 설정할 수 있습니다.

### 5. 랜딩페이지 삭제하기

1. 랜딩페이지 목록에서 "컨텐츠관리" → 삭제 아이콘 클릭
2. 삭제 확인
3. 삭제 완료

## 📊 그룹별 관리

### 그룹 만들기
- 페이지 생성 시 "페이지 그룹" 필드에 그룹 이름 입력
- 예: "16 B2B크루즈", "15 B2B테스트", "14 [B2C] 크루즈"

### 그룹 표시
- 그룹별로 자동으로 그룹화되어 표시
- 그룹 헤더에 그룹 이름과 페이지 개수 표시
- 그룹 펼치기/접기 가능

### 그룹 정렬
- 그룹 번호 순으로 정렬 (내림차순)
- 예: 16 → 15 → 14 → 13...

## 🔗 바로가기 URL 작동 방식

### 1. URL 생성
```
랜딩페이지 ID → 짧은 코드 생성 → /i/코드 형식 URL 저장
```

### 2. URL 접근
```
사용자가 /i/코드 접근 → 코드로 랜딩페이지 찾기 → 실제 랜딩페이지로 리다이렉트
```

### 3. 재생성
- "신규생성" 버튼으로 새 코드 생성
- 기존 URL은 무효화됨
- 카카오톡 캐시 문제 해결에 유용
