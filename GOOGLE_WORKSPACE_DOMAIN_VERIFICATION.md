# Google Workspace 도메인 인증 정보

## 인증 코드 (TXT 레코드)

### 방법 1: TXT 레코드 (권장)

**이름/호스트:**
- `@` 또는 비워두기 (기본값으로 설정)

**값/데이터:**
```
google-site-verification=DwKIt9M7nfdXEiGWv3fgg1EECHPbPNHM7YTgkagp_hQ
```

**TTL:**
- 기본값 또는 최저값으로 설정

---

### 방법 2: CNAME 레코드 (대체 방법)

**이름/호스트:**
```
wzfr7l2kapol
```

**값/데이터:**
```
gv-7oikacln7z4cjj.dv.googlehosted.com
```

**TTL:**
- 기본값 또는 최저값으로 설정

---

## DNS 설정 방법

### 1. 도메인 호스트 사이트 접속
- 가비아, 후이즈, 네임칩, AWS Route 53 등

### 2. DNS 관리 메뉴로 이동
- "DNS 관리", "DNS 설정", "네임서버 관리" 등

### 3. 레코드 추가
- 레코드 타입: `TXT` (방법 1) 또는 `CNAME` (방법 2)
- 위의 정보 입력
- 저장

### 4. 확인
- DNS 변경 반영: 몇 분 ~ 최대 48시간
- Google Workspace 관리 콘솔에서 "확인" 클릭

---

## 중요 사항

⚠️ **주의:**
- TXT 레코드 값은 정확히 복사해야 합니다 (공백 포함)
- DNS 변경 후 반영까지 시간이 걸릴 수 있습니다
- 인증 완료 후 공유 드라이브를 만들 수 있습니다

---

## 인증 완료 후 다음 단계

1. 공유 드라이브 생성
2. 폴더 구조 생성
3. 서비스 계정 권한 설정
4. 폴더 ID 확인 및 .env 설정
