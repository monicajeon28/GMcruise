## 🚀 샘플 상품 생성 방법

### 방법 1: npm 스크립트 사용 (권장)

```bash
npm run create:sample-products
```

### 방법 2: 직접 실행

```bash
npx tsx scripts/create-sample-products.ts
```

## 📋 생성되는 상품 정보

각 샘플 상품에는 다음 정보가 포함됩니다:

- **기본 정보**
  - 상품 코드 (SAMPLE-001 ~ SAMPLE-015)
  - 크루즈 라인
  - 선박명
  - 패키지명
  - 박수/일수
  - 기본 가격
  - 설명

- **일정 정보**
  - 일정 패턴 (itineraryPattern)
  - 각 일자별 항구 방문 정보

- **태그 정보**
  - 인기 상품 (isPopular)
  - 추천 상품 (isRecommended)
  - 프리미엄 상품 (isPremium)
  - 일본 상품 (isJapan)
  - 경제 상품 (isBudget)
  - 긴급 상품 (isUrgent)

## ✏️ 상품 편집 방법

### 관리자 페이지에서 편집

1. **관리자 로그인**
   ```
   /admin/login
   ```

2. **상품 관리 페이지 접속**
   ```
   /admin/products
   ```

3. **상품 선택 및 편집**
   - 목록에서 원하는 상품 클릭
   - 상세 정보 수정
   - 저장

### 수동 등록

1. **새 상품 등록**
   - `/admin/products` 페이지에서 "새 상품 등록" 버튼 클릭
   - 필수 정보 입력:
     - 상품 코드
     - 크루즈 라인
     - 선박명
     - 패키지명
   - 선택 정보 입력:
     - 박수/일수
     - 기본 가격
     - 설명
     - 일정 패턴
     - 태그 (인기, 추천, 프리미엄 등)

## 🔍 확인 사항

### 생성 후 확인

1. **상품 목록 확인**
   ```

   ```

   - 1줄당 5개, 최대 3줄 표시 확인

3. **상품 상세 페이지 확인**
   - 각 상품 클릭 시 상세 페이지 정상 표시 확인
