# 여권 링크 테스트 가이드

## 📍 대리점장 대시보드에서 여권 링크 확인 방법

### 1. 신규 예약 등록 후 확인

**경로**: `/partner/[partnerId]/reservation/new`

**절차**:
1. 대리점장 대시보드 접속
2. "신규 예약 등록" 메뉴 클릭
3. 결제 내역 선택 및 예약 정보 입력
4. 예약 생성 완료 후 **여권 등록 링크 섹션** 확인

**기능**:
- ✅ **링크 복사**: 클립보드에 링크 복사
- ✅ **미리보기**: 새 창(1200x800)에서 고객 화면 미리보기
- ✅ **새 창에서 열기**: 새 탭에서 여권 등록 폼 열기

### 2. 예약 성공 화면 구성

예약 생성 완료 후 다음 섹션이 표시됩니다:

```
✅ 예약 생성이 완료되었습니다
   ↓
🛂 여권 등록 링크
   - 링크 URL 표시
   - [링크 복사] 버튼
   - [미리보기] 버튼 (새 창 1200x800)
   - [새 창에서 열기] 버튼
   ↓
[대시보드로 돌아가기] 버튼
```

### 3. 미리보기 기능

**미리보기 버튼 클릭 시**:
- 새 창(1200x800 크기)에서 여권 등록 폼이 열림
- 고객이 보는 화면과 동일한 UI 확인 가능
- 단계별 입력 (1-5단계) 확인 가능
- 예약 정보 자동 로드 확인 가능

### 4. 링크 형식

```
https://[도메인]/customer/passport/[reservationId]
```

예시:
```
https://cruisedot.co.kr/customer/passport/123
```

### 5. 테스트 체크리스트

- [ ] 예약 생성 후 여권 링크 섹션이 표시되는지 확인
- [ ] 링크 복사 버튼이 정상 작동하는지 확인
- [ ] 미리보기 버튼 클릭 시 새 창이 열리는지 확인
- [ ] 미리보기 창에서 단계별 입력(1-5단계)이 표시되는지 확인
- [ ] 예약 정보(상품명, 출발일 등)가 자동으로 로드되는지 확인
- [ ] 고객 화면과 동일한 UI인지 확인

### 6. 접근 경로 요약

**대리점장 대시보드**:
```
/partner/[partnerId]/dashboard
  → 신규 예약 등록 클릭
  → /partner/[partnerId]/reservation/new
  → 예약 생성 완료
  → 여권 등록 링크 섹션 확인
```

### 7. 추가 확인 사항

**고객 화면과 동일한지 확인**:
- Step 1: 예약 정보 확인 (구매한 상품 정보 표시)
- Step 2: 대표자 정보 입력
- Step 3: 동행자 정보 입력
- Step 4: 여권 정보 입력 (OCR 스캔 포함)
- Step 5: 최종 확인 및 제출

**예약 정보 자동 로드 확인**:
- 상품명, 크루즈 라인, 선박명
- 일정 (박수/일수)
- 출발일
- 총 인원
- 예약자 정보
