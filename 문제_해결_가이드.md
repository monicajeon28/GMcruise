# 문제 해결 가이드

## 1. 터미널 에러 해결

### 문제
```
npm error path /home/userhyeseon28/projects/package.json
npm error enoent Could not read package.json
```

### 해결 방법

터미널에서 다음 명령어를 실행하세요:

```bash
cd /home/userhyeseon28/projects/cruise-guide
npm run dev
```

**중요:** 반드시 `cruise-guide` 폴더 안에서 실행해야 합니다!

---

## 2. Prisma 클라이언트 에러 해결

### 문제
`audioFileGoogleDriveUrl` 필드를 찾을 수 없음

### 해결 방법

1. **Prisma 클라이언트 재생성** (이미 완료)
```bash
cd /home/userhyeseon28/projects/cruise-guide
npx prisma generate
```

2. **개발 서버 재시작** (필수!)
```bash
npm run dev
```

3. **브라우저 새로고침**
- `Ctrl+Shift+R` (하드 리프레시)

---

## 3. 판매 목록이 표시되지 않는 문제

### 원인
- Prisma 클라이언트가 최신 스키마를 반영하지 않음
- 서버가 재시작되지 않음

### 해결 방법

1. 서버 재시작 (위의 방법 참고)
2. 브라우저 새로고침
3. 여전히 문제가 있으면:
```bash
rm -rf .next
npm run dev
```

---

## 확인 사항

서버 재시작 후:
- ✅ "내 판매 목록" 섹션에 4개의 판매가 표시되어야 함
- ✅ 각 판매의 상태가 올바르게 표시되어야 함
- ✅ "확정 요청" 버튼이 작동해야 함
