## 1. 마케팅 픽셀 설정

### 1-1. Google Analytics 4 (GA4) 설정

#### Step 1: Google Analytics 계정 만들기
1. https://analytics.google.com 접속
2. Google 계정으로 로그인
3. "측정 시작" 클릭
4. 계정 이름 입력 (예: "크루즈닷")
5. 속성 이름 입력 (예: "크루즈닷 웹사이트")
6. 보고 시간대 선택: 한국 시간
7. 통화 선택: KRW (원)
8. "만들기" 클릭

#### Step 2: 측정 ID 확인하기
1. Google Analytics 대시보드에서
2. 왼쪽 하단 ⚙️ (관리) 클릭
3. "속성" → "데이터 스트림" 클릭
4. 웹 스트림 클릭
5. **측정 ID** 복사 (예: `G-XXXXXXXXXX`)

#### Step 3: 관리자 페이지에 입력하기
1. 웹사이트 관리자 페이지 접속: `https://cruisedot.co.kr/admin/settings`
2. "마케팅 설정" 탭 클릭
3. **Google Analytics 4 (GA4) 측정 ID** 입력란에 복사한 ID 붙여넣기
4. "Google 픽셀 활성화" 체크박스 ✅ 체크
5. "저장하기" 버튼 클릭

---

### 1-2. Google Tag Manager 설정 (선택사항)

#### Step 1: Google Tag Manager 계정 만들기
1. https://tagmanager.google.com 접속
2. "계정 만들기" 클릭
3. 계정 이름 입력 (예: "크루즈닷")
4. 국가 선택: 대한민국
5. 컨테이너 이름 입력 (예: "크루즈닷 웹사이트")
6. 타겟 플랫폼: "웹" 선택
7. "만들기" 클릭

#### Step 2: 컨테이너 ID 확인하기
1. Google Tag Manager 대시보드에서
2. 컨테이너 ID 확인 (예: `GTM-XXXXXXX`)
3. 복사

#### Step 3: 관리자 페이지에 입력하기
1. 관리자 페이지 → "마케팅 설정" 탭
2. **Google Tag Manager ID** 입력란에 붙여넣기
3. "저장하기" 클릭

---

### 1-3. Facebook Pixel 설정

#### Step 1: Facebook 이벤트 관리자 접속
1. https://business.facebook.com 접속
2. Facebook 비즈니스 계정으로 로그인
3. 왼쪽 메뉴에서 "이벤트 관리자" 클릭
4. "데이터 소스" → "웹" 클릭
5. "Facebook Pixel" 선택
6. "Pixel 만들기" 클릭

#### Step 2: Pixel ID 확인하기
1. Pixel 만들기 완료 후
2. Pixel ID 확인 (예: `1234567890123456`)
3. 복사

#### Step 3: 관리자 페이지에 입력하기
1. 관리자 페이지 → "마케팅 설정" 탭
2. **Facebook Pixel ID** 입력란에 붙여넣기
3. "Facebook 픽셀 활성화" 체크박스 ✅ 체크
4. "저장하기" 클릭

---

### 1-4. 네이버 픽셀 설정 (선택사항)

#### Step 1: 네이버 검색광고 접속
1. https://searchad.naver.com 접속
2. 네이버 계정으로 로그인
3. "도구" → "전환 추적" 클릭
4. "웹사이트 전환 추적" 선택
5. Pixel ID 생성 및 확인

#### Step 2: 관리자 페이지에 입력하기
1. 관리자 페이지 → "마케팅 설정" 탭
2. **네이버 픽셀 ID** 입력란에 입력
3. "네이버 픽셀 활성화" 체크박스 ✅ 체크
4. "저장하기" 클릭

---

### 1-5. 카카오 픽셀 설정 (선택사항)

#### Step 1: 카카오 비즈보드 접속
1. https://bizboard.kakao.com 접속
2. 카카오 계정으로 로그인
3. "전환 추적" → "웹사이트 전환" 클릭
4. Pixel ID 생성 및 확인

#### Step 2: 관리자 페이지에 입력하기
1. 관리자 페이지 → "마케팅 설정" 탭
2. **카카오 픽셀 ID** 입력란에 입력
3. "카카오 픽셀 활성화" 체크박스 ✅ 체크
4. "저장하기" 클릭

## 2. SEO 설정

### 2-1. 기본 SEO 설정 (홈페이지)

#### Step 1: SEO 관리 페이지 접속
1. 관리자 페이지 접속: `https://cruisedot.co.kr/admin/seo`
2. "새 SEO 설정 추가" 버튼 클릭

#### Step 2: 홈페이지 SEO 설정 입력
1. **페이지 경로**: `/` 입력
2. **페이지 타입**: "홈" 선택
3. **페이지 제목**: `크루즈 가이드 - AI 여행 도우미` (60자 이내)
4. **메타 설명**: `AI 가이드 지니와 함께하는 특별한 크루즈 여행. 크루즈 상품 예약, 여행 준비, 실시간 안내까지 모든 것을 한 곳에서.` (150-160자)
5. **키워드**: `크루즈, 크루즈 여행, AI 여행 도우미, 크루즈 가이드, 크루즈 상품, 크루즈 예약, 일본 크루즈, 해외 크루즈` (쉼표로 구분)
6. **OG 이미지 URL**: `https://cruisedot.co.kr/images/ai-cruise-logo.png`
7. **OG URL**: `https://cruisedot.co.kr`
8. **Robots**: "index, follow (기본)" 선택
9. "저장하기" 클릭

2. **페이지 타입**: "카테고리" 선택
3. **페이지 제목**: `크루즈 상품 목록 - 크루즈 가이드`

5. **키워드**: `크루즈 상품, 크루즈 예약, 크루즈 여행, 일본 크루즈, 동남아 크루즈, 지중해 크루즈`

### 2-3. 개별 상품 페이지 SEO 설정 (자동)

> **참고**: 개별 상품 페이지(`/products/[productCode]`)는 자동으로 SEO 메타데이터가 생성됩니다.
> 
> - 상품명이 자동으로 제목에 포함됩니다
> - 상품 설명이 메타 설명에 사용됩니다
> - 상품 이미지가 OG 이미지로 사용됩니다
> 
> 추가로 커스터마이징이 필요하면 `/admin/seo`에서 해당 상품의 경로로 설정을 추가할 수 있습니다.

## 3. 설정 확인 방법

### 3-1. 마케팅 픽셀 작동 확인

#### Google Analytics 확인
1. Google Analytics 대시보드 접속
2. "실시간" 보고서 확인
3. 웹사이트를 방문하면 실시간으로 방문자 수가 표시됩니다

#### Facebook Pixel 확인
1. Facebook 이벤트 관리자 접속
2. "테스트 이벤트" 클릭
3. 웹사이트 URL 입력
4. 웹사이트에서 페이지를 이동하면 이벤트가 표시됩니다

---

### 3-2. SEO 설정 확인

#### 메타 태그 확인
1. 웹사이트 페이지에서 우클릭 → "페이지 소스 보기" (또는 F12)
2. `<head>` 태그 안에 다음이 있는지 확인:
   - `<title>` 태그
   - `<meta name="description">` 태그
   - `<meta property="og:title">` 태그
   - `<meta property="og:image">` 태그

#### Google Search Console 확인
1. https://search.google.com/search-console 접속
2. 사이트 등록 (아직 안 했다면)
3. "URL 검사" 도구로 페이지 검색
4. Google이 페이지를 어떻게 보는지 확인

## 4. 자주 묻는 질문 (FAQ)

### Q1: 픽셀 ID를 어디서 찾나요?
**A**: 각 플랫폼의 관리자 페이지에서 확인할 수 있습니다:
- Google Analytics: 관리 → 속성 → 데이터 스트림
- Facebook: 이벤트 관리자 → 데이터 소스
- 네이버: 검색광고 → 도구 → 전환 추적
- 카카오: 비즈보드 → 전환 추적

### Q2: 픽셀을 활성화했는데 데이터가 안 보여요
**A**: 
1. 픽셀 ID가 정확한지 확인
2. "활성화" 체크박스가 체크되어 있는지 확인
3. 웹사이트를 새로고침하고 몇 분 기다려보기
4. 브라우저 개발자 도구(F12) → Console에서 오류 확인

### Q3: SEO 설정을 여러 페이지에 해야 하나요?
**A**: 
- 주요 페이지(홈, 상품 목록, 커뮤니티)만 설정해도 됩니다
- 개별 상품 페이지는 자동으로 SEO 메타데이터가 생성됩니다
- 추가로 최적화하고 싶은 페이지가 있으면 `/admin/seo`에서 추가 설정 가능

### Q4: OG 이미지는 어떤 크기가 좋나요?
**A**: 
- 권장 크기: **1200 x 630 픽셀**
- 형식: JPG 또는 PNG
- 파일 크기: 1MB 이하 권장
- 이미지 URL은 전체 URL(https://로 시작)로 입력

### Q5: 키워드는 몇 개가 적당한가요?
**A**: 
- 3-5개 정도가 적당합니다
- 쉼표로 구분하여 입력
- 너무 많으면 오히려 역효과가 날 수 있습니다
