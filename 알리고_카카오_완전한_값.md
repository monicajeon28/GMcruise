# 알리고 카카오톡 완전한 환경변수 값

관리자 패널에 이미 저장되어 있는 알리고 카카오톡 관련 환경변수의 완전한 값입니다.

---

## 📋 완전한 값

### ALIGO_KAKAO_SENDER_KEY

```
13b13496a0f51e9a602706d0dd8b27598088dd5a
```

- **설명**: 알리고 카카오톡 발신 키 (Sender Key)
- **용도**: 카카오톡 알림톡/친구톡 발송 시 사용
- **발급 방법**: 알리고 (https://www.aligo.in) → 카카오톡 발신 키 발급

### ALIGO_KAKAO_CHANNEL_ID

```
cruisedot
```

- **설명**: 알리고에 등록된 카카오 채널 ID
- **용도**: 카카오톡 메시지 발송 시 채널 연결
- **확인 방법**: 알리고 관리자 페이지 → 카카오톡 설정

---

## 📝 Vercel 환경변수 설정

Vercel에서 설정할 때 다음과 같이 입력하세요:

```bash
# 알리고 카카오톡 발신 키
ALIGO_KAKAO_SENDER_KEY=13b13496a0f51e9a602706d0dd8b27598088dd5a

# 알리고 카카오 채널 ID
ALIGO_KAKAO_CHANNEL_ID=cruisedot
```

---

## ✅ 관리자 패널에서 확인/수정

이 값들은 이미 관리자 패널 (`/admin/settings`)에서 확인하고 수정할 수 있습니다:

1. **관리자 패널 접속**: `/admin/settings`
2. **SMS API 설정 섹션**에서 확인:
   - "카카오 채널 Senderkey" → `ALIGO_KAKAO_SENDER_KEY`
   - "카카오 채널 ID" → `ALIGO_KAKAO_CHANNEL_ID`
3. **수정하기 버튼** 클릭하여 값을 수정할 수 있습니다
4. **저장하기** 클릭 시 자동으로 Vercel 환경변수에 동기화됩니다

---

## 🔄 자동 동기화

관리자 패널에서 값을 수정하고 저장하면:

1. ✅ 로컬 환경변수 (`.env.local`)에 저장
2. ✅ Vercel 환경변수에 자동 동기화 (VERCEL_API_TOKEN 설정 시)
3. ✅ 변경사항 즉시 반영

**주의**: Vercel 자동 동기화를 사용하려면 `VERCEL_API_TOKEN`과 `VERCEL_PROJECT_ID`가 설정되어 있어야 합니다.
