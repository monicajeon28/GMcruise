### 3. 교육 페이지 (별도 도메인) ⭐⭐⭐⭐⭐

**구조**:
```
education.cruisedot.com (별도 도메인)
  ├─ 네이버 아카데미 스타일
  ├─ 썸네일 기반 선택
  ├─ 카테고리별 분류
  │   ├─ 마케팅 교육
  │   ├─ 세일즈 교육
  │   └─ 인솔 교육
  └─ 비디오 플레이어
```

**접근**:
- 관리자: `/admin/education` 또는 별도 도메인
- 대리점장: 별도 도메인 또는 링크
- 판매원: 별도 도메인 또는 링크

---

### 4. 판매원 대시보드 (계약 완료 시 자동 생성) ⭐⭐⭐⭐⭐

**구조**:
```
/partner/[agentId]/dashboard
  ├─ 나만의 SNS 마케팅 링크
  ├─ 나만의 고객 관리
  ├─ 교육 페이지 링크
  ├─ 문자 CRM
  └─ 판매 관리
```

**자동 생성 프로세스**:
```
계약서 완료
  ↓
판매원 대시보드 자동 생성
  ↓
대리점장 대시보드에 [판매원 이름] 버튼 추가
```

---

### 5. 대리점장 대시보드 ⭐⭐⭐⭐⭐
