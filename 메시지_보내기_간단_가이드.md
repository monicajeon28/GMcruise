# 메시지 보내기 간단 가이드

> **작성일**: 2025년 1월 28일  
> **대상**: 판매원, 대리점장

---

## 📱 메시지 보내는 방법 (판매원/대리점장)

### 1단계: 대시보드 접속
- **판매원**: `/admin/affiliate/agent-dashboard`
- **대리점장**: `/admin/affiliate/team-dashboard`

### 2단계: Team Messages 버튼 클릭
- 대시보드 우측 상단 또는 빠른 메뉴에서 **"Team Messages"** 버튼 찾기
- 버튼에 미읽음 개수가 빨간 배지로 표시됨 (예: "3")

### 3단계: 메시지 모달에서 "보내기" 버튼 클릭
- 메시지 모달이 열리면 상단 오른쪽에 **"보내기"** 버튼이 있음
- **"보내기"** 버튼 클릭

### 4단계: 메시지 작성
1. **수신자 선택** (드롭다운)
   - 판매원: 담당 대리점장, 관리자만 표시
   - 대리점장: 소속 판매원, 다른 대리점장, 관리자 표시
2. **제목 입력**
3. **내용 입력**

### 5단계: 전송
- **"보내기"** 버튼 클릭
- 성공 메시지 확인

---

## 🔍 메시지 확인하는 방법

### 받은 메시지 확인
1. **"Team Messages"** 버튼 클릭
2. **"Received"** 탭 확인
3. 메시지 클릭하여 읽음 처리

### 보낸 메시지 확인
1. **"Team Messages"** 버튼 클릭
2. **"Sent"** 탭 확인
3. 본인이 보낸 메시지 목록 확인
4. 삭제 버튼으로 메시지 삭제 가능

---

## ⚠️ 주의사항

### 판매원
- ✅ 담당 대리점장에게만 메시지 전송 가능
- ✅ 관리자에게 메시지 전송 가능
- ❌ 다른 판매원에게는 메시지 전송 불가
- ❌ 본인에게는 메시지 전송 불가

### 대리점장
- ✅ 소속 판매원에게 메시지 전송 가능
- ✅ 다른 대리점장에게 메시지 전송 가능
- ✅ 관리자에게 메시지 전송 가능
- ❌ 본인에게는 메시지 전송 불가

---

## 🎯 빠른 테스트 시나리오

### 테스트 1: 판매원 → 대리점장
1. 판매원 로그인
2. 대시보드 → "Team Messages" → "보내기"
3. 담당 대리점장 선택 → 제목/내용 입력 → 전송
4. 대리점장 로그인 → "Team Messages" → "Received" 탭에서 확인

### 테스트 2: 대리점장 → 판매원
1. 대리점장 로그인
2. 대시보드 → "Team Messages" → "보내기"
3. 소속 판매원 선택 → 제목/내용 입력 → 전송
4. 판매원 로그인 → "Team Messages" → "Received" 탭에서 확인

### 테스트 3: 보낸 메시지 확인
1. 메시지 전송 후
2. "Team Messages" → "Sent" 탭 클릭
3. 방금 보낸 메시지 확인
4. 삭제 버튼으로 삭제 가능
