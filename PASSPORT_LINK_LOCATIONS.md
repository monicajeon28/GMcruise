# 대리점장 대시보드 - 여권 링크 확인 위치 가이드

## 📍 여권 링크를 확인할 수 있는 위치

### 1. 수동여권등록 페이지 (예약 생성 후)

**경로**: `/partner/[partnerId]/reservation/new`

**접근 방법**:
1. 대리점장 대시보드 접속
2. **"수동여권등록"** 카드 클릭
3. 결제 내역 선택 및 예약 정보 입력
4. 예약 생성 완료 후 → **"여권 등록 링크" 섹션** 표시

**기능**:
- ✅ **링크 복사**: 클립보드에 링크 복사
- ✅ **미리보기**: 새 창(1200x800)에서 고객 화면 미리보기
- ✅ **새 창에서 열기**: 새 탭에서 여권 등록 폼 열기

**링크 형식**:
```
https://[도메인]/customer/passport/[reservationId]
```

---

### 2. 고객 상세 페이지 (여권 보내기 모달)

**경로**: `/partner/[partnerId]/customers`

**접근 방법**:
1. 대리점장 대시보드 접속
2. **"고객 관리"** 메뉴 클릭
3. 고객 목록에서 고객 선택
4. 고객 상세 화면에서 **"여권 보내기"** 버튼 클릭
5. **"여권 보내기" 모달** 열림

**기능**:
- ✅ **링크 복사**: 여권 업로드 링크 복사
- ✅ **미리보기**: 새 창(1200x800)에서 고객 화면 미리보기
- ✅ **알리고 API 발송**: 알리고 API로 직접 문자 발송

**링크 형식**:
```
https://[도메인]/api/public/passport-upload?leadId=[leadId]&partnerId=[partnerId]
```

---

### 3. 여권 요청 관리 페이지

**경로**: `/partner/[partnerId]/passport-requests`

**접근 방법**:
1. 대리점장 대시보드 접속
2. **"여권 요청 관리"** 메뉴 클릭 (있는 경우)
3. 여권 제출 링크 생성 및 관리

---

## 🎯 빠른 확인 방법

### 방법 1: 수동여권등록에서 확인 (추천)

```
대시보드 → 수동여권등록 클릭 → 예약 생성 → 여권 링크 섹션 확인
```

### 방법 2: 고객 관리에서 확인

```
대시보드 → 고객 관리 → 고객 선택 → 여권 보내기 버튼 → 모달에서 링크 확인
```

---

## 📋 미리보기 기능 사용법

### 미리보기 버튼 클릭 시:
1. 새 창(1200x800 크기)이 열립니다
2. 고객이 보는 여권 등록 폼이 표시됩니다
3. 단계별 입력(1-5단계)을 확인할 수 있습니다:
   - Step 1: 예약 정보 확인
   - Step 2: 대표자 정보 입력
   - Step 3: 동행자 정보 입력
   - Step 4: 여권 정보 입력
   - Step 5: 최종 확인 및 제출

### 확인 사항:
- [ ] 예약 정보가 자동으로 로드되는지
- [ ] 구매한 상품 정보가 표시되는지
- [ ] 단계별 진행이 정상 작동하는지
- [ ] 고객 화면과 동일한 UI인지

## 🔗 링크 형식 차이

### 수동여권등록에서 생성되는 링크
- **형식**: `/customer/passport/[reservationId]`
- **용도**: 예약 기반 여권 등록
- **특징**: 예약 정보 자동 로드, 단계별 입력 UI

### 고객 관리에서 생성되는 링크
- **형식**: `/api/public/passport-upload?leadId=[leadId]&partnerId=[partnerId]`
- **용도**: 고객(Lead) 기반 여권 업로드
- **특징**: 고객 정보 기반, 본사로 직접 전송
