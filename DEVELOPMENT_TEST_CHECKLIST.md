**작성일**: 2025-11-23  
**목적**: 최적화 작업 후 실제 동작 확인

## 📋 테스트 항목

### 1. 관리자 대시보드 테스트 (`/admin/dashboard`)

#### ✅ 확인 사항
- [ ] 페이지가 정상적으로 로드되는가?
- [ ] 차트가 표시되는가? (로딩 후)
- [ ] 차트 로딩 중 스피너/플레이스홀더가 표시되는가?
- [ ] 차트 데이터가 올바르게 표시되는가?
- [ ] 다른 통계 카드들이 정상적으로 표시되는가?
- [ ] 새로고침 시에도 정상 작동하는가?

#### 🔍 확인 방법
1. 개발 서버 실행: `npm run dev`
2. 관리자로 로그인
3. `/admin/dashboard` 접속
4. 차트가 로드될 때까지 대기 (1-2초)
5. 차트가 정상적으로 표시되는지 확인

#### ⚠️ 예상되는 현상
- **정상**: 차트가 1-2초 후 표시됨 (동적 임포트로 인한 지연)
- **비정상**: 차트가 전혀 표시되지 않음 → 에러 확인 필요

---

### 2. 메인 페이지 테스트 (`/`)

#### ✅ 확인 사항
- [ ] 페이지가 정상적으로 로드되는가?
- [ ] YouTube 컴포넌트들이 표시되는가?
  - [ ] YouTube Shorts 슬라이더
  - [ ] YouTube Live 섹션
  - [ ] YouTube Videos 슬라이더
- [ ] 프로모션 배너 캐러셀이 표시되는가?
- [ ] 로딩 중 플레이스홀더가 표시되는가?
- [ ] 스크롤 시 컴포넌트가 지연 로드되는가?

#### 🔍 확인 방법
1. 메인 페이지 접속 (`/`)
2. 페이지 하단으로 스크롤
3. YouTube 컴포넌트들이 로드되는지 확인
4. 네트워크 탭에서 지연 로드 확인

#### ⚠️ 예상되는 현상
- **정상**: 초기 로딩이 빨라지고, YouTube 컴포넌트는 스크롤 시 로드
- **비정상**: 컴포넌트가 전혀 표시되지 않음 → 에러 확인 필요

---

### 3. 빌드 최적화 확인

#### ✅ 확인 사항
- [ ] 빌드가 성공적으로 완료되는가?
- [ ] 번들 크기가 감소했는가?
- [ ] 초기 로딩 속도가 개선되었는가?

#### 🔍 확인 방법
1. **번들 분석 실행**:
   ```bash
   npm run analyze
   ```
2. 브라우저에서 번들 분석 결과 확인
3. recharts가 별도 청크로 분리되었는지 확인
4. YouTube 컴포넌트가 별도 청크로 분리되었는지 확인

#### 📊 확인할 지표
- **초기 번들 크기**: 이전 대비 감소했는가?
- **recharts 청크**: 별도로 분리되었는가?
- **YouTube 컴포넌트 청크**: 별도로 분리되었는가?

---

### 4. 성능 측정

#### ✅ 확인 사항
- [ ] 페이지 로딩 시간
- [ ] Time to Interactive (TTI)
- [ ] First Contentful Paint (FCP)

#### 🔍 확인 방법
1. Chrome DevTools 열기 (F12)
2. Network 탭에서 "Disable cache" 체크
3. Performance 탭에서 "Record" 클릭
4. 페이지 새로고침
5. 성능 메트릭 확인

#### 📊 목표 지표
- **초기 로딩**: 2-3초 이내
- **TTI**: 3-4초 이내
- **FCP**: 1-2초 이내

## 🐛 에러 확인 방법

### 브라우저 콘솔 확인
1. 개발자 도구 열기 (F12)
2. Console 탭 확인
3. 에러 메시지 확인

### 네트워크 탭 확인
1. Network 탭 열기
2. 페이지 새로고침
3. 실패한 요청 확인 (빨간색)
4. 404 에러 확인

### React DevTools 확인
1. React DevTools 설치 (필요시)
2. Components 탭에서 컴포넌트 트리 확인
3. 에러가 있는 컴포넌트 확인

## 🚨 문제 발생 시 대응

### 차트가 표시되지 않는 경우
1. 브라우저 콘솔 에러 확인
2. `components/admin/DashboardCharts.tsx` 파일 확인
3. 동적 임포트 경로 확인

### 컴포넌트가 로드되지 않는 경우
1. 네트워크 탭에서 청크 로드 확인
2. 컴포넌트 경로 확인
3. 동적 임포트 설정 확인

### 빌드 실패하는 경우
1. 에러 메시지 확인
2. 변경된 파일들 확인
3. 롤백 고려
