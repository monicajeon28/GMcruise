# 🧪 테스트 계정 정보

## 📋 파트너 모드 로그인 (대리점장/판매원)

### 대리점장 (Boss)
- **아이디**: `boss1`, `boss2`, `boss3`... (숫자만 증가)
- **비밀번호**: `1101`
- **로그인 페이지**: `/community/login` 또는 `/mall/login`

### 판매원 (User)
- **아이디**: `user1`, `user2`, `user3`... (숫자만 증가)
- **비밀번호**: `1101`
- **로그인 페이지**: `/community/login` 또는 `/mall/login`

### 파트너 로그인 방법
1. `/community/login` 또는 `/mall/login` 페이지 접속
2. **아이디** 필드에 `boss1` 또는 `user1` 입력
3. **비밀번호** 필드에 `1101` 입력
4. 로그인 버튼 클릭

**참고**: 
- 파트너 아이디는 이름 없이 숫자만 증가합니다 (`boss1`, `boss2`, `user1`, `user2`...)
- `mallUserId` 필드로도 검색 가능
- `phone` 필드로도 검색 가능
- 대소문자 구분 없음
- 로그인 API에서 `mode: 'partner'` 또는 비밀번호 `qwe1` 입력 시 자동으로 파트너 모드로 인식됩니다

## 🔑 비밀번호 관리

### 비밀번호 표시
- 관리자 패널에서 어필리에이트 프로필 편집 시 **현재 비밀번호가 문자 그대로 표시**됩니다
- 프로필 편집 화면의 "파트너 계정" 섹션에서 확인 가능

### 비밀번호 변경
- 판매원이 그만두면 본사에서 비밀번호를 변경할 수 있습니다
- 프로필 편집 화면에서 "비밀번호 변경" 필드에 새 비밀번호 입력
- 비워두면 비밀번호 변경 안 함
- 비밀번호 변경 이벤트가 자동으로 기록됩니다

## ⚠️ 로그인이 안 될 경우

### 파트너 로그인 문제 해결
1. **아이디 확인**: `boss1`, `user1` (대소문자 구분 없음, 이름 없이 숫자만)
2. **비밀번호 확인**: `1101` (정확히 입력)
3. **모드 확인**: 파트너 모드로 로그인하는지 확인
   - `/community/login` 사용
   - 또는 비밀번호에 `qwe1` 입력 시 자동 파트너 모드
4. **계정 생성 확인**: 테스트 계정이 DB에 있는지 확인
   ```bash
   # 스크립트 실행하여 테스트 계정 생성
   npm run ts-node scripts/create-affiliate-test-data.ts
   ```

### 관리자 로그인 문제 해결
1. **전화번호 확인**: `01024958013` 또는 `01038609161` (정확히 입력)
2. **비밀번호 확인**: `0313` (정확히 입력)
3. **역할 확인**: `role: 'admin'`인지 확인
4. **계정 생성 확인**: 관리자 계정이 DB에 있는지 확인
   ```bash
   # 스크립트 실행하여 관리자 계정 생성/수정
   npm run ts-node scripts/create-admin.ts
   # 또는
   npm run ts-node scripts/create-admin-accounts.ts
   ```

## 📝 로그인 API 확인

- **요청 본문**:
  ```json
  {
    "phone": "boss1",
    "password": "1101",
    "name": "boss1",
    "mode": "partner"
  }
  ```
