## 🎯 마케팅 기능 분석

### 1. **마케팅 대시보드** (`/admin/marketing/dashboard`)
- **상태**: ✅ 완성
- **기능**: 
  - 전체 유입/전환 통계
  - 랜딩페이지별 분석
  - 기간별 통계 (오늘/이번주/이번달)
  - 실시간 갱신 (1분마다)
- **데이터 표시**: 
  - 전체 유입, 전환, 전환율, 활성 페이지
  - 랜딩페이지별 유입/전환/전환율
- **판단**: 잘 작동하며 유용함, **유지 필요**

### 2. **마케팅 플랜 설정** (`/admin/marketing/accounts`)
- **상태**: ⚠️ 기능은 완성이나 사용 빈도 낮음
- **기능**:
  - 플랜 관리 (BASIC, STANDARD, PREMIUM, EXPERT, ENTERPRISE)
  - 고객/페이지/퍼널 제한 설정
- **문제점**: 
  - 실제로 플랜을 자주 변경할 일이 없음
  - UI는 잘 되어있으나 실용성 낮음
- **판단**: **필요시 접근 제한** 또는 **설정 페이지로 이동** 권장

### 3. **리드 관리** (`/admin/marketing/leads`)
- **상태**: ✅ 완성
- **기능**:
  - 리드 목록 조회
  - 상태별 필터 (NEW, CONTACTED, QUALIFIED, CONVERTED, LOST)
  - 리드 스코어 표시
- **판단**: 기능적으로 완성되었으나, **어필리에이트 고객 관리와 중복**될 수 있음

### 4. **퍼널 관리** (`/admin/marketing/funnels`)
- **상태**: ⚠️ 미완성
- **기능**:
  - 퍼널 생성/수정/삭제
  - 퍼널 분석 기능 존재
- **문제점**:
  - 퍼널 상세 분석 페이지 (`/admin/marketing/funnels/[id]`) 미구현
  - 퍼널 편집 페이지 (`/admin/marketing/funnels/[id]/edit`) 미구현
  - 실제 퍼널 데이터가 활용되는지 불명확
- **판단**: **미완성 기능으로 배포 전 완성 필요** 또는 **숨김 처리**

### 5. **고객 관리** (`/admin/marketing/customers`)
- **상태**: ✅ 완성
- **기능**:
  - 고객 검색/필터 (그룹별, 날짜별)
  - 엑셀 업로드/다운로드
  - 그룹문자 발송
  - 휴대폰번호 일괄 등록
  - 고객 상세정보 및 기록 조회
- **판단**: **매우 유용하고 완성도 높음**, **유지 필요**

### 6. **랜딩 페이지 관리** (`/admin/marketing/pages`)
- **상태**: ⚠️ 통합됨 - 사용 중단 권장
- **기능**:
  - 랜딩페이지 목록
  - 통계 조회
  - 공개/비공개 전환
- **문제점**:
  - 페이지 상단에 "⚠️ 이 페이지는 통합되었습니다: 랜딩페이지 관리는 `/admin/landing-pages`에서 이용하세요" 경고 표시
  - 실제로는 `/admin/landing-pages`를 사용해야 함
- **판단**: **중복 기능**, **삭제 또는 리다이렉트 처리 권장**

### 7. **튜토리얼** (`/admin/marketing/tutorial`)
- **상태**: ✅ 완성
- **기능**:
  - 9단계 마케팅 튜토리얼
  - 초등학생도 이해 가능한 설명
- **판단**: 사용자 온보딩에 유용, **유지**

## 🎨 UI/UX 개선 제안

### 1. **메뉴 구조 정리**
현재 마케팅과 어필리에이트가 별도로 나뉘어 있는데, 사용자 입장에서 혼란스러울 수 있음.

**제안:**
```
관리자 패널
├── 📊 대시보드
│   ├── 마케팅 대시보드
│   ├── 판매원 활동 대시보드
│   └── 정산 대시보드
├── 👥 고객 관리
│   ├── 마케팅 자동화 고객
│   └── 어필리에이트 리드
├── 🤝 어필리에이트
│   ├── 인력 관리
│   ├── 판매 확정 승인
│   ├── 정산 대시보드
│   └── 지급명세서
├── 📄 랜딩 페이지 (통합)
├── ⚙️ 설정
│   ├── 마케팅 플랜
│   └── 기타 설정
└── ❓ 도움말
    └── 튜토리얼
```

### 2. **중복 네이밍 명확화**
- "고객 관리" → "마케팅 고객 관리" / "어필리에이트 리드 관리"
- "대시보드" → 각각 명확한 이름 부여
