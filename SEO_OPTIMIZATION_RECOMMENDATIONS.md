## ✅ 적용 방법

1. 관리자 패널 접속: `/admin/settings`
2. "SEO 전역 설정" 섹션으로 스크롤
3. "기본 SEO 설정" 섹션에서 위 값들을 입력
4. "저장하기" 클릭
5. 완료!

## 🚀 추가 최적화 팁

1. **콘텐츠 최적화**
   - 각 페이지에 타겟 키워드를 자연스럽게 포함
   - H1, H2 태그에 키워드 사용
   - 내부 링크에 관련 키워드 사용

2. **이미지 최적화**
   - 모든 이미지에 의미 있는 alt 태그 추가
   - 이미지 파일명에 키워드 포함 (예: `일본-크루즈-여행.jpg`)

3. **페이지 속도 최적화**
   - 이미지 최적화 (WebP, AVIF 형식 사용)
   - 코드 스플리팅
   - 캐싱 전략

4. **외부 SEO (백링크)**
   - 소셜 미디어 공유
   - 블로그 포스팅
   - 파트너 사이트 링크
