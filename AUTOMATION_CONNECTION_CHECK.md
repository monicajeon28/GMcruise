# 관리자 패널-파트너 대시보드 자동화 연결 상태 체크

> 작성일: 2025-01-28  
> 목적: 관리자 패널과 대리점장/판매원 대시보드 간 자동화 연결 상태 확인

---

## 📋 체크 항목

### 1. 계약서 관리 자동화 ✅/❌

#### 관리자 패널 (`/admin/affiliate/contracts`)
- [ ] 계약서 승인 버튼 → 자동화 확인
- [ ] 계약서 거부 버튼 → 자동화 확인
- [ ] 계약서 완료 처리 → 자동화 확인
- [ ] 계약서 PDF 생성 및 전송 → 자동화 확인
- [ ] 계약서 승인 후 프로필 자동 생성 → 확인 필요

#### 파트너 대시보드 (`/partner/[partnerId]/dashboard`)
- [ ] 계약서 상태 실시간 반영 → 확인 필요
- [ ] 계약서 완료 후 대시보드 접근 권한 → 확인 필요

### 2. 고객 체크 기능 자동화 ✅/❌

#### 관리자 패널 (`/admin/affiliate/customers`)
- [ ] 고객 목록 조회 → 자동화 확인
- [ ] 고객 상태 변경 → 자동화 확인
- [ ] 고객 상세 정보 확인 → 자동화 확인

#### 파트너 대시보드 (`/partner/[partnerId]/customers`)
- [ ] 고객 목록 조회 → 자동화 확인
- [ ] 고객 등록 → 자동화 확인
- [ ] 고객 상태 업데이트 → 자동화 확인

### 3. 판매 실적 확인 자동화 ✅/❌

#### 관리자 패널 (`/admin/affiliate/adjustments`)
- [ ] 판매 실적 조회 → 자동화 확인
- [ ] 구매 완료 승인 → 자동화 확인
- [ ] 수당 조정 승인 → 자동화 확인

#### 파트너 대시보드 (`/partner/[partnerId]/dashboard`)
- [ ] 판매 실적 조회 → 자동화 확인
- [ ] 판매 확정 요청 → 자동화 확인
- [ ] 판매 상태 확인 → 자동화 확인

### 4. 판매원 체크 기능 자동화 ✅/❌

#### 관리자 패널 (`/admin/affiliate/profiles`)
- [ ] 판매원 목록 조회 → 자동화 확인
- [ ] 판매원 정보 수정 → 자동화 확인
- [ ] 판매원-대리점장 관계 확인 → 자동화 확인

#### 파트너 대시보드 (`/partner/[partnerId]/dashboard`)
- [ ] 담당 멘토 정보 표시 → 자동화 확인
- [ ] 판매원-대리점장 관계 표시 → 자동화 확인

---

## 🔍 발견된 문제점

### 즉시 확인 필요
1. 계약서 승인 후 자동 프로필 생성 여부
2. 계약서 완료 처리 후 대시보드 접근 권한 부여
3. 판매 확정 요청 후 자동 수당 계산
4. 고객 상태 변경 시 실시간 반영

### 개선 필요
1. 버튼 클릭 한 번으로 모든 자동화 처리
2. 상태 변경 시 실시간 알림
3. 데이터 동기화 확인

---
