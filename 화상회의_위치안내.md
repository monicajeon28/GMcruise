# 📍 화상 회의 기능 위치 안내

## ⚠️ 중요: 현재 보고 계신 페이지

이미지를 보니 **"예약 메시지 관리"** 페이지에 계시는 것 같습니다.
화상 회의 기능은 **"대시보드"** 페이지에 있습니다!

## 🎯 화상 회의 기능 위치

### 관리자 패널

**현재 위치**: `/admin/scheduled-messages` (예약 메시지 관리)
**화상 회의 위치**: `/admin/dashboard` (대시보드) ⭐

**접근 방법**:
1. 왼쪽 사이드바에서 **"기본 메뉴"** 섹션 클릭
2. **"대시보드"** 메뉴 클릭
3. 대시보드 페이지 최상단에 "화상 회의" 섹션이 있습니다

**직접 URL**:
```
http://localhost:3000/admin/dashboard
```

### 대리점장 대시보드

**접근 경로**: `/partner/[대리점장ID]/dashboard`

**예시**:
```
http://localhost:3000/partner/boss1/dashboard
```

## 🔍 화면에서 찾는 방법

### 관리자 패널

1. **왼쪽 사이드바 확인**
   - "기본 메뉴" 섹션
   - 첫 번째 메뉴: **"대시보드"** 📊

2. **대시보드 페이지 접속**
   - "대시보드" 클릭

3. **화상 회의 섹션 확인**
   - 페이지 최상단
   - 헤더 바로 아래
   - 파란색 배경 카드
   - "📹 화상 회의" 제목
   - 오른쪽에 "화상 회의" 버튼

### 대리점장 대시보드

1. **대시보드 접속**
   - `/partner/[ID]/dashboard`

2. **화상 회의 섹션 확인**
   - "개인 링크" 섹션 아래
   - "빠른 메뉴" 섹션 위
   - 파란색 배경 카드

## ⚠️ 에러 해결

### "미팅방 생성에 실패했습니다" 에러

**해결 완료**:
1. ✅ 데이터베이스 마이그레이션 실행 (`npx prisma db push`)
2. ✅ Prisma 클라이언트 재생성

**다음 단계**:
1. 서버 재시작
   ```bash
   cd /home/userhyeseon28/projects/cruise-guide
   # 기존 서버 종료 (Ctrl+C)
   npm run dev:all
   ```

2. 브라우저 새로고침
   - Ctrl+Shift+R (하드 리프레시)
   - 또는 브라우저 캐시 삭제

3. 올바른 페이지 접속
   - 관리자: `/admin/dashboard` (대시보드)
   - 대리점장: `/partner/[ID]/dashboard`
