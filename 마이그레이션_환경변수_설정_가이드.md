## 📝 설정 방법

### 1. .env.local 파일 열기

```bash
# 프로젝트 루트 디렉토리에서
nano .env.local
# 또는
code .env.local
```

### 2. 위 환경변수들을 복사하여 붙여넣기

**⚠️ 주의사항**:
- `GOOGLE_DRIVE_SERVICE_ACCOUNT_PRIVATE_KEY`는 여러 줄로 입력해야 합니다
- Private Key의 전체 내용을 복사하세요 (BEGIN부터 END까지)

### 3. 저장 후 확인

```bash
npm run migrate:check-env
```

---

## ✅ 설정 완료 후

환경변수 설정이 완료되면 다음 명령어로 확인하세요:

```bash
npm run migrate:check-env
```

모든 환경변수가 설정되어 있으면 마이그레이션을 시작할 수 있습니다!
