# 판매원 대시보드 메시지 확인 위치

> **작성일**: 2025년 1월 28일

---

## 📍 메시지 확인 위치

### 1. 판매원 대시보드 접속
- **URL**: `/admin/affiliate/agent-dashboard`
- 판매원 계정으로 로그인 후 접속

### 2. 페이지 상단 우측에 "팀 메시지" 버튼

```
┌─────────────────────────────────────────────────────────┐
│  판매원 대시보드                                         │
│  판매원별 리드 진행, 판매 실적, 수당 및...              │
│                                                         │
│  [팀 메시지 🔴3]  [새로고침]  ← 여기!                   │
└─────────────────────────────────────────────────────────┘
```

**위치 설명**:
- 페이지 최상단 헤더 영역
- "새로고침" 버튼 바로 왼쪽
- 연한 청록색 배경 (`bg-teal-50`)
- 청록색 테두리 (`border-teal-200`)
- 청록색 텍스트 (`text-teal-700`)

### 3. 미읽음 개수 배지
- 미읽음 메시지가 있으면 버튼 우측 상단에 **빨간 원형 배지** 표시
- 예: `[팀 메시지 🔴3]` (3개 미읽음)
- 9개 이상이면 `9+`로 표시

### 4. 버튼 클릭 시
- 메시지 모달이 열림
- 받은 메시지/보낸 메시지 탭으로 구분
- 상단에 "보내기" 버튼으로 새 메시지 작성 가능

## 🎯 단계별 안내

### Step 1: 판매원 대시보드 접속
1. 판매원 계정으로 로그인
2. `/admin/affiliate/agent-dashboard` 접속

### Step 2: "팀 메시지" 버튼 찾기
- 페이지 최상단 우측
- "새로고침" 버튼 왼쪽
- 연한 청록색 배경의 버튼

### Step 3: 버튼 클릭
- 클릭하면 메시지 모달이 열림
- 받은 메시지/보낸 메시지 탭 확인

### Step 4: 메시지 확인
- **"Received"** 탭: 받은 메시지 목록
- **"Sent"** 탭: 보낸 메시지 목록

## 🔍 버튼 특징

### 시각적 특징
- **배경색**: 연한 청록색 (`bg-teal-50`)
- **테두리**: 청록색 (`border-teal-200`)
- **텍스트**: 진한 청록색 (`text-teal-700`)
- **아이콘**: 메시지 아이콘 (`FiMessageSquare`)
- **호버 효과**: 배경색이 조금 더 진해짐 (`hover:bg-teal-100`)

### 기능
- 클릭 시 메시지 모달 열기
- 미읽음 개수 실시간 표시
- 5분마다 자동 갱신

## ⚠️ 찾기 어려운 경우

### 문제 1: 버튼이 보이지 않음
**원인**:
- 페이지가 완전히 로드되지 않음
- 브라우저 캐시 문제

**해결**:
1. 페이지 새로고침 (F5 또는 Ctrl+R)
2. 브라우저 캐시 삭제 후 재접속
3. 개발자 도구 (F12)에서 콘솔 에러 확인

### 문제 2: 미읽음 개수가 표시되지 않음
**원인**:
- 실제로 미읽음 메시지가 없음
- API 호출 실패

**해결**:
1. 버튼은 여전히 클릭 가능 (미읽음이 없어도 버튼은 표시됨)
2. 개발자 도구에서 네트워크 탭 확인
3. `/api/admin/affiliate/my-messages` API 호출 확인
