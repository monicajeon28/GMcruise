# 마케팅 자동화 기능 완성도 가이드

> 작성일: 2025-01-28  
> 목적: 구현된 모든 마케팅 자동화 기능을 초등학생도 이해할 수 있게 설명

---

## 🎯 구현 완료된 기능 목록

### ✅ 1. 관리계정 (MarketingAccount)
**무엇인가요?** 여러 개의 마케팅 계정을 만들 수 있어요!

**어디서 사용하나요?**
- `/admin/marketing/accounts` 페이지에서 관리해요
- 프리미엄 플랜은 최대 10개까지 만들 수 있어요

**어떻게 작동하나요?**
1. "계정 생성" 버튼을 눌러요
2. 계정 이름과 코드를 입력해요
3. 각 계정마다 고객, 페이지, 퍼널을 따로 관리할 수 있어요

**예시:** "A회사 계정", "B회사 계정"처럼 나눠서 관리해요!

---

### ✅ 2. 고객 관리 (MarketingCustomer)
**무엇인가요?** 고객 정보를 저장하고 관리할 수 있어요!

**어디서 사용하나요?**
- `/admin/marketing/customers` 페이지에서 관리해요
- 프리미엄 플랜은 최대 9,999명까지 저장할 수 있어요

**어떻게 작동하나요?**
1. "고객 추가" 버튼을 눌러요
2. 이름, 전화번호, 이메일, 출처를 입력해요
3. 고객 목록에서 검색, 필터링, 수정, 삭제가 가능해요
4. 엑셀로 다운로드해서 백업할 수 있어요
5. 자동 백업 기능도 있어요

**특별한 기능:**
- ✅ 고객 추가
- ✅ 고객 삭제
- ✅ 엑셀 다운로드
- ✅ 자동 백업
- ✅ 고객 그룹에 추가하기
- ✅ 메시지 보내기

**예시:** "홈페이지에서 온 고객", "전화로 문의한 고객" 등으로 구분해요!

---

### ✅ 3. 퍼널 관리 (MarketingFunnel)
**무엇인가요?** 고객이 구매까지 가는 길을 만들어요!

**어디서 사용하나요?**
- `/admin/marketing/funnels` 페이지에서 관리해요
- 프리미엄 플랜은 최대 40개까지 만들 수 있어요

**어떻게 작동하나요?**
1. "퍼널 생성" 버튼을 눌러요
2. 퍼널 이름과 설명을 입력해요
3. 각 단계에서 고객이 무엇을 해야 하는지 정해요
4. 퍼널을 활성화/비활성화할 수 있어요

**예시:** "신규 고객 퍼널", "재구매 고객 퍼널" 등으로 나눠서 만들 수 있어요!

---

### ✅ 4. 리드 관리 (MarketingLead)
**무엇인가요?** 구매하고 싶어하는 고객을 찾아요!

**어디서 사용하나요?**
- `/admin/marketing/leads` 페이지에서 관리해요

**어떻게 작동하나요?**
1. 랜딩 페이지나 광고를 통해 리드를 모아요
2. 각 리드마다 점수를 매겨서 우선순위를 정해요
3. 점수가 높은 리드부터 먼저 연락하면 더 잘 팔려요

**예시:** 홈페이지에서 "무료 상담 신청"을 한 사람이 리드예요!

---

### ✅ 5. 리드 스코어링 (LeadScore)
**무엇인가요?** 어떤 고객이 더 구매할 가능성이 높은지 점수를 매겨요!

**어떻게 작동하나요?**
- 0점부터 100점까지 점수를 매겨요
- 점수가 높을수록 구매할 가능성이 높아요
- 예: 홈페이지를 많이 본 고객 = 높은 점수
- 예: 상품을 장바구니에 넣은 고객 = 매우 높은 점수

**예시:** 80점 이상 = 매우 관심 많음, 60점 이상 = 관심 있음, 40점 이하 = 관심 적음

---

### ✅ 6. 리드 상호작용 (LeadInteraction)
**무엇인가요?** 고객과 자동으로 연락할 수 있어요!

**어떻게 작동하나요?**
- 고객이 특정 행동을 하면 자동으로 메시지가 보내져요
- 예: 상품을 본 고객 → 1일 후 자동으로 할인 정보 보내기
- 예: 장바구니에 넣은 고객 → 3일 후 자동으로 구매 유도 메시지
- 문자, 이메일, 카카오톡으로 보낼 수 있어요

**예시:** "상품을 봤지만 구매 안 한 고객"에게 자동으로 할인 쿠폰을 보내요!

---

### ✅ 7. 퍼널 전환 (FunnelConversion)
**무엇인가요?** 고객이 실제로 구매했는지 확인해요!

**어떻게 작동하나요?**
- 고객이 구매하면 "전환"이라고 해요
- 몇 명이 구매했는지, 얼마를 벌었는지 볼 수 있어요
- 전환율(구매한 사람 비율)도 계산해줘요

**예시:** 100명이 방문했고 10명이 구매 = 10% 전환율

---

### ✅ 8. 재구매 추적 (RepeatPurchase)
**무엇인가요?** 한 번 구매한 고객이 다시 구매하도록 유도해요!

**어떻게 작동하나요?**
- 이미 구매한 고객은 다시 구매할 가능성이 높아요
- 구매 후 일정 시간이 지나면 자동으로 재구매 메시지를 보내요
- 예: 구매 후 3개월 후 → "새로운 상품이 나왔어요!" 메시지

**예시:** 크루즈 여행을 다녀온 고객에게 6개월 후 새로운 크루즈 상품을 추천해요!

---

### ✅ 9. 바이럴 루프 (ViralLoop)
**무엇인가요?** 고객이 친구를 추천하도록 만들어요!

**어떻게 작동하나요?**
- 고객이 친구를 추천하면 둘 다 혜택을 받아요
- 예: 추천한 고객 = 10% 할인, 추천받은 고객 = 5% 할인
- 고객이 자발적으로 친구를 불러와서 더 많이 팔려요
- 추천 코드를 만들어서 쉽게 공유할 수 있어요

**예시:** "친구 초대하면 둘 다 할인!" 같은 이벤트를 만들 수 있어요!

## 🔗 기존 기능과의 연결

### ✅ 고객 그룹 관리 연결
- 마케팅 고객을 기존 고객 그룹에 추가할 수 있어요
- 고객 목록에서 "고객 그룹에 추가" 버튼을 누르면 연결돼요

### ✅ 고객 메시지 연결
- 마케팅 고객에게 직접 메시지를 보낼 수 있어요
- 고객 목록에서 "메시지 보내기" 버튼을 누르면 연결돼요

### ✅ 예약 메시지 연결
- 마케팅 고객을 예약 메시지 대상으로 설정할 수 있어요
- 고객 그룹에 추가하면 자동으로 예약 메시지 대상이 돼요

## 💾 백업 및 다운로드

### 엑셀 다운로드
- 고객 목록을 엑셀 파일로 다운로드할 수 있어요
- "엑셀 다운로드" 버튼을 누르면 바로 다운로드돼요

### 자동 백업
- 고객 데이터를 자동으로 백업해요
- "백업하기" 버튼을 누르면 JSON 파일로 저장돼요
- `backups/marketing-customers/` 폴더에 저장돼요
