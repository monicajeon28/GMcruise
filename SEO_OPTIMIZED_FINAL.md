## 📊 키워드 분석

### 상위 노출 확률이 높은 이유

1. **장문 키워드 (Long-tail Keywords) 우선**
   - "크루즈 예약" > "크루즈" (경쟁 적음)
   - "일본 크루즈 예약" > "크루즈" (더 구체적)
   - 검색 의도가 명확하여 전환율 높음

2. **검색 의도별 분류**
   - **정보성**: 준비, 가이드, 후기, 팁, 정보
   - **거래성**: 예약, 가격, 일정, 패키지
   - **지역성**: 일본, 지중해, 알래스카, 부산, 인천

3. **경쟁도 분석**
   - 1순위 키워드: 경쟁 적음, 검색 의도 명확
   - 2순위 키워드: 경쟁 적음, 지역 특화
   - 3순위 키워드: 브랜드 검색, 특정 선사 관심
   - 4순위 키워드: 정보 검색, 경쟁 적음
   - 5순위 키워드: 경쟁 치열, 높은 검색량

## 🎯 SEO 최적화 전략

### 1. 키워드 밀도 최적화
- 각 페이지에 타겟 키워드를 자연스럽게 포함
- H1, H2 태그에 우선순위 높은 키워드 사용
- 내부 링크에 관련 키워드 사용

### 2. 콘텐츠 최적화
- **예약 페이지**: "크루즈 예약", "크루즈 패키지 예약" 키워드 집중
- **일정 페이지**: "크루즈 일정", "크루즈 여행 일정" 키워드 집중
- **가격 페이지**: "크루즈 가격", "크루즈 여행 가격" 키워드 집중
- **지역별 페이지**: "일본 크루즈", "지중해 크루즈" 등 지역 키워드 집중

### 3. 메타데이터 최적화
- 각 페이지별 고유한 title과 description
- 타겟 키워드를 title 앞부분에 배치
- Description에 행동 유도 문구 포함

### 4. 구조화된 데이터
- Product 스키마 (크루즈 상품)
- Organization 스키마 (크루즈닷지니AI)
