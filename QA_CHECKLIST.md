## 📋 체크리스트 사용 방법

1. **기능 추가/수정 후**: 이 체크리스트의 해당 섹션 확인
2. **배포 전**: 전체 체크리스트 확인
3. **문제 발견 시**: 체크리스트에 항목 추가

## 🔐 1. 인증 및 보안

### 1.1 로그인/로그아웃
- [ ] 일반 사용자 로그인 정상 작동
- [ ] 관리자 로그인 정상 작동
- [ ] 파트너 로그인 정상 작동
- [ ] 로그아웃 정상 작동
- [ ] 세션 만료 시 자동 로그아웃
- [ ] 잘못된 비밀번호 입력 시 에러 메시지 표시
- [ ] Rate Limiting 작동 확인 (너무 많은 시도 시 차단)

### 1.2 권한 관리
- [ ] 관리자만 관리자 패널 접근 가능
- [ ] 일반 사용자는 관리자 패널 접근 불가
- [ ] 파트너는 자신의 대시보드만 접근 가능
- [ ] 권한 없는 페이지 접근 시 403 에러 또는 리다이렉트

### 1.3 보안
- [ ] `.env` 파일이 Git에 포함되지 않음
- [ ] 민감한 정보가 코드에 하드코딩되지 않음
- [ ] CSRF 토큰 검증 작동
- [ ] SQL Injection 방지 (Prisma 사용 확인)
