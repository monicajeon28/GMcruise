# 🚀 이미지 라이브러리 시스템 빠른 시작 가이드
**작성일**: 2025-11-23

---

## ✅ 완료된 기능

### 1. 이미지 업로드 및 WebP 자동 변환
- 이미지 업로드 시 자동으로 WebP 변환
- 원본과 WebP 버전 모두 저장
- 폴더별 정리 가능

### 2. 이미지 라이브러리 페이지
- **URL**: `/admin/images`
- 갤러리 형태로 이미지 표시
- 검색 기능
- 폴더별 관리

### 3. 소스코드 자동 생성 및 복사
- Next.js Image 태그
- HTML img 태그
- 이미지 URL만

---

## 📖 사용 방법

### 1단계: 이미지 업로드
1. 관리자 페이지 접속
2. 사이드바에서 "이미지 라이브러리" 클릭
3. "이미지 업로드" 버튼 클릭
4. 이미지 파일 선택 (여러 개 가능)
5. 자동으로 WebP 변환됨 ✅

### 2단계: 소스코드 복사
1. 업로드된 이미지 클릭
2. 모달에서 원하는 형식 선택:
   - **Next.js Image 태그**: Next.js 프로젝트용
   - **HTML img 태그**: 일반 HTML용
   - **이미지 URL**: URL만 필요할 때
3. "복사" 버튼 클릭
4. 코드가 클립보드에 복사됨 ✅

### 3단계: 코드 사용
1. Visual Editor 또는 코드 편집기에서
2. 복사한 코드 붙여넣기
3. 완성! 🎉

## 📁 폴더 구조

```
public/
  image-library/
    images/              # 기본 폴더
      image1.jpg
      image1.webp       # 자동 변환됨
      image2.png
      image2.webp
    크루즈사진/          # 사용자 생성 폴더
      코스타세레나/
        객실1.jpg
        객실1.webp
```
