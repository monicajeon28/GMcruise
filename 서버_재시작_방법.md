# 서버 재시작 방법

## 문제
터미널에서 `npm run dev` 실행 시 에러 발생

## 해결 방법

### 1. 프로젝트 폴더로 이동

터미널에서 다음 명령어 실행:

```bash
cd /home/userhyeseon28/projects/cruise-guide
```

### 2. 서버 재시작

```bash
npm run dev
```

## 전체 명령어

```bash
cd /home/userhyeseon28/projects/cruise-guide
npm run dev
```

---

## 참고

- 현재 위치 확인: `pwd`
- 프로젝트 폴더 확인: `ls` (cruise-guide 폴더가 보여야 함)
