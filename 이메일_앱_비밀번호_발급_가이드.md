# 🔑 Gmail 앱 비밀번호 발급 가이드

이메일 발송 기능을 사용하기 위해 Google 계정에서 "앱 비밀번호"를 발급받아야 합니다.

## 📋 준비사항

- Google 계정 (jmonica@cruisedot.co.kr)
- 2단계 인증이 활성화되어 있어야 함

---

## 🎯 단계별 발급 방법

### 1단계: Google 계정에 로그인

1. 브라우저에서 https://myaccount.google.com 접속
2. `jmonica@cruisedot.co.kr` 계정으로 로그인

---

### 2단계: 2단계 인증 활성화 (아직 안 했다면)

1. 왼쪽 메뉴에서 **"보안"** 클릭
2. **"Google에 로그인"** 섹션 찾기
3. **"2단계 인증"** 클릭
4. 화면 안내에 따라 2단계 인증 설정 완료
   - 보통 휴대폰 번호로 인증 코드를 받거나
   - Google Authenticator 앱 사용

> ⚠️ **중요**: 2단계 인증이 꺼져있으면 앱 비밀번호를 발급받을 수 없습니다!

---

### 3단계: 앱 비밀번호 발급

#### 방법 1: 검색으로 찾기 (가장 빠름)

1. Google 계정 페이지 상단 검색창 클릭
2. **"앱 비밀번호"** 또는 **"app password"** 검색
3. 검색 결과에서 **"앱 비밀번호"** 클릭

#### 방법 2: 직접 찾기

1. **"보안"** 탭 클릭
2. **"Google에 로그인"** 섹션 스크롤
3. **"2단계 인증"** 아래에 **"앱 비밀번호"** 링크 클릭

---

### 4단계: 앱 비밀번호 생성

1. **"앱 선택"** 드롭다운 클릭
2. **"기타(맞춤 이름)"** 선택
3. 텍스트 입력창에 **"Nodemailer"** 또는 **"크루즈닷 이메일"** 입력
4. **"만들기"** 버튼 클릭

---

### 5단계: 16자리 비밀번호 복사

1. 화면에 **16자리 영문/숫자 조합**이 표시됩니다
   - 예시: `abcd efgh ijkl mnop`
   - 띄어쓰기 없이: `abcdefghijklmnop`

2. **전체 16자리를 복사** (띄어쓰기 제외)

3. ⚠️ **중요**: 이 비밀번호는 **한 번만 표시**됩니다!
   - 다시 볼 수 없으므로 **지금 복사**하세요

---

### 6단계: .env 파일에 설정

프로젝트 루트의 `.env` 또는 `.env.local` 파일을 열고:

```bash
# 이메일 발송 설정
EMAIL_USER=jmonica@cruisedot.co.kr
EMAIL_PASS=abcdefghijklmnop
```

> 💡 `EMAIL_PASS`에 복사한 16자리를 붙여넣으세요 (띄어쓰기 없이!)

---

## ✅ 확인 방법

1. 서버 재시작 (환경 변수 적용)
2. 관리자 패널에서 문서를 생성하고 이메일 발송 테스트
3. 발송 성공 시 설정 완료! 🎉

## 🆘 문제 해결

### "앱 비밀번호를 찾을 수 없어요"
- 2단계 인증이 활성화되어 있는지 확인
- "보안" → "앱 비밀번호" 경로로 다시 확인

### "이메일 발송이 실패해요"
- `.env` 파일에 올바르게 입력했는지 확인 (띄어쓰기 없이!)
- 서버를 재시작했는지 확인
- Gmail 계정에 로그인이 정상적으로 되는지 확인

### "2단계 인증을 켤 수 없어요"
- Google 계정 보안 설정 확인
- 회사 Google Workspace 정책 확인 (관리자에게 문의)

## 📝 참고사항

- **앱 비밀번호는 일반 비밀번호와 다릅니다**
  - 일반 비밀번호: 로그인할 때 사용
  - 앱 비밀번호: 외부 앱(예: Nodemailer)에서 사용

- **보안상 안전합니다**
  - 앱 비밀번호는 해당 앱에서만 사용 가능
  - 필요 시 언제든지 취소 가능

- **여러 개 만들 수 있습니다**
  - 필요하면 여러 개의 앱 비밀번호를 만들 수 있음
  - 각각 다른 이름으로 구분 가능 (예: "Nodemailer", "개발서버", "운영서버")
